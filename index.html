Addressing a Critical Need

Understanding the challenge and our innovative approach to solving it

The Problem

Existing navigation systems for visually impaired individuals are often bulky, conspicuous, and uncomfortable to use. These devices stand out significantly, causing self-consciousness and limiting mobility. Traditional white canes, while helpful, cannot detect obstacles at various heights or provide advanced warnings.


Many assistive devices are heavy, require handheld operation, or draw unwanted attention, making users feel stigmatized rather than empowered.

Our Solution

An ergonomic, discreet solution seamlessly integrated into everyday footwear. Our smart shoes provide natural obstacle detection without drawing attention, allowing users to navigate confidently while maintaining their personal style and dignity.


The system is lightweight, requires no handheld components, and operates silently through haptic feedback, creating a truly invisible assistive technology.

How the System Works

A comprehensive IoT solution combining sensors, intelligent processing, and adaptive feedback

Obstacle Detection

Ultrasonic sensors and LIDAR technology continuously scan the path ahead, detecting obstacles from 15cm to 150cm away with 95% accuracy. The system identifies objects at various heights, from ground-level hazards to head-level obstacles.

Adaptive Signal System

The ESP32 microcontroller processes sensor data in real-time and generates adaptive feedback signals. As obstacles get closer, vibration intensity increases proportionally, providing intuitive distance awareness without visual confirmation.

GPS Emergency Tracking

Integrated GPS NEO-6M module provides precise location tracking with ±5 meter accuracy. In case of emergency, the system can transmit the user's exact location to designated contacts or emergency services via WiFi or Bluetooth connectivity.

Complete assistive technology ecosystem: Integration of sensors, microcontrollers, wireless connectivity, and multi-modal feedback systems for comprehensive navigation support

Technical Approach and System Workflow

Sophisticated integration of hardware and software for seamless operation

Intelligent Processing Pipeline

1

Environmental Sensing

Ultrasonic HC-SR04 (40kHz) and LIDAR TFmini Plus sensors continuously scan a 120-degree field of view ahead, capturing real-time distance data at millisecond intervals.

2

Data Processing

ESP32-WROVER-B dual-core processor (240MHz) analyzes sensor streams using adaptive algorithms, filtering noise and classifying obstacles by distance and urgency.

3

Location Tracking

GPS NEO-6M module maintains continuous position awareness, logging coordinates and enabling emergency location broadcasting via wireless connectivity.

4

Feedback Generation

Multi-modal alert system delivers context-aware feedback through vibration motors, piezo buzzer, and optional speaker, adapting to environmental conditions and user preferences.

Key System Components


ESP32 Microcontroller

The brain of our system, the ESP32-WROVER-B features dual-core processing at 240MHz, 520KB SRAM, and 4MB Flash storage. It handles real-time sensor data acquisition, processing, and control logic while managing wireless communications.


Ultrasonic Distance Sensor

HC-SR04 ultrasonic sensor provides accurate distance measurements from 15cm to 150cm using 40kHz sound waves. Ideal for detecting nearby obstacles and uneven surfaces.


LIDAR Sensor

TFmini Plus LIDAR offers precise long-range detection with high accuracy, capable of detecting smaller objects and providing detailed environmental mapping for enhanced safety.

Multi-sensor approach: Combining ultrasonic and LIDAR technologies for comprehensive obstacle detection capabilities

Advanced Features


GPS Navigation Module

NEO-6M GPS module with ±5 meter accuracy provides reliable outdoor positioning. Supports emergency location sharing and route tracking for enhanced safety and independence.


Feedback Systems

Piezo buzzer (80Hz-16kHz range) and three directional vibration motors provide non-visual feedback. Sound and vibration patterns adapt based on obstacle distance, creating an intuitive warning system.


Wireless Connectivity

WiFi 802.11 b/g/n and Bluetooth 5.0 enable mobile app integration, remote monitoring, firmware updates, and emergency communication capabilities.

ESP32 ecosystem integration: Wireless connectivity, sensor fusion, and real-time data processing architecture

Project Development Timeline

Systematic approach from concept to deployment

Phase 1: Hardware Setup

Connect and integrate ESP32 microcontroller with ultrasonic sensors, LIDAR, GPS module, and feedback systems (buzzer and vibration motors). Design and assemble the physical prototype with proper power management.

Phase 2: System Programming

Develop firmware for ESP32 focusing on sensor data acquisition, real-time processing algorithms, and control logic. Implement wireless communication protocols and emergency alert systems.

Phase 3: Testing and Validation

Comprehensive testing of obstacle detection accuracy, signal adaptive response, and GPS tracking reliability. Field testing in various real-world environments to validate system performance.

Phase 4: Deployment and Refinement

Integrate all components into final prototype form factor. Conduct user testing with visually impaired individuals, gather feedback, and refine the system for optimal comfort and effectiveness.
